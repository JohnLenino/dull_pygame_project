1. Введение

Целью данного проекта является разработка игры 2048 с использованием библиотеки Pygame.

2. Основной функционал

В рамках данного проекта был реализован следующий функционал:

Создание игрового поля размером 4x4 с возможностью адаптации размера поля в 8x8 и 16x16.
Генерация случайных блоков с числовыми значениями (2 или 4) на пустых клетках игрового поля.
Возможность движения блоков в четыре направления: вверх, вниз, влево и вправо.
Объединение блоков одинаковых значений при слиянии.
Отображение текущего счета игрока.
Отображение сообщения об окончании игры при возможности совершить ход.
3. Структура проекта

Проект структурирован следующим образом:

button.py: Модуль, содержащий класс кнопок для меню и окна поражения.
const.py: Файл с различными игровыми константами, такими как цвета.
controller.py: Модуль, отвечающий за обработку действий игрока (нажатие на стрелочки и кнопочки).
data.txt: Файл, в котором хранится рекорд и поле, на котором этот рекорд был установлен.
logic.py: Модуль, где производится работа с матрицами numpy как с полем игры.
requirements.txt: Файл, содержащий необходимые библиотеки (numpy и pygame).
run.py: Основной файл, в котором запускается вся игра.
sounds/: Папка со звуками для игры.
view.py: Модуль, отвечающий за отрисовку поля.
4. Использование Pygame

Для создания графического интерфейса и управления игровым процессом использовалась библиотека Pygame. Она позволяет легко реализовать отображение графики, обработку событий клавиатуры и мыши, а также управление анимацией.

5. Выводы

В результате работы над проектом была успешно реализована игра 2048 с использованием библиотеки Pygame. Проект демонстрирует наши навыки в работе с pygame.
